CSS

The CSS file defines styles for various elements in the HTML document, such as the font family and background color of the body, the box shadow and padding of the header, the font size and margin of the headings, and so on. By separating the content from the style, we can more easily maintain and update the appearance of our webpage without having to change the underlying

Usage of CSS

CSS can be added to HTML documents in 3 ways:

- Inline - by using the style attribute inside HTML elements

- Internal - by using a <style> element in the <head> section

- External - by using a <link> element to link to an external CSS file

The most common way to add CSS, is to keep the styles in external CSS files. However, in this website we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

Incline CSS

Incline CSS is a method of applying CSS styling directly to an HTML element

Uses style attribute of an HTML element.

<!DOCTYPE html>
        <h1 style=color > content </h1>
        <p style=color > content </p>

- The style attribute can include various style properties, such as color, which sets the text color of an element.
