<Table>

, HTML tables allow web developers to arrange data into rows and columns.

In this example, the <table> section contains several key elements: 

<!DOCTYPE html>
    <html>
      <head>
        <code>
      </head>
      <body>
        <h1> heading </h1>
        <p> text content who want to write </p >
        <img  image  >
        <table>
         <th>
           <tr>
            <th> table head</th>
         </th>
         <tr>
          <td> table cell </td>
         </tr>
         </td>
        </table>
    </html>

- <tr>, <thead> mean table row

- <td>, <tbody> mean table data

- <th>, <tfoot> mean table header
